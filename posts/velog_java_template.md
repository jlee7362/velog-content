## ✍️ 코드 설명  

- `Scanner sc = new Scanner(System.in);`  
  → 사용자 입력을 받기 위한 표준 방식

- `Integer.valueOf(a, 16);`  
  → 문자열을 16진수로 해석하여 정수로 변환

- `System.out.printf("%X", num);`  
  → 정수를 16진수 대문자로 출력

## ✅ 정리  

- `Integer.valueOf(s, 진수)` → 문자열 진법 처리 가능  
- `printf("%X")` → 16진수 포맷 출력  
- `Scanner`로 입력 받아 `int`로 변환하면 다양한 계산에 활용 가능
